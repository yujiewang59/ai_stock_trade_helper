## 1. skill基本信息

| 输入要求 | 股票基础信息|
|---------|--------------------------|
| 输出格式 | 结构化决策结果：决策类型（买入/持有/卖出）、置信度、决策原因 |

## 2. 输入参数定义

  'ADX指数',
  '趋势强度',
  'EMA8',
  'EMA21',
  'EMA55',
  '短期趋势',
  '中期趋势',
  'Z评分',
  '价格相对布林带位置',
  'RSI14',
  'RSI28',
  '布林带上轨',
  '布林带下轨',
  '1月动量',

  '成交量确认度',

### 2.1 必选参数

| 参数名 | 类型 | 说明 |
|-------|------|------|
| `stock_info` | Dict | 股票基础信息，包含：
- `stock_name`: 股票名称
- `stock_code`: 股票代码
- `industry`: 所属行业（支持：消费 / 科技 / 医药 / 周期 / 金融 / 房地产 / 公用事业，其他行业使用通用模型）
| `indicators` | Dict | 技术面指标键值对，支持的核心指标包括：<br/>**趋势分析维度**：`adx`(ADX指数), `trend_strength`(趋势强度), `ema_8`(8日EMA), `ema_21`(21日EMA), `ema_55`(55日EMA), `short_term_trend`(短期趋势), `medium_term_trend`(中期趋势)<br/>**均值回归维度**：`z_score`(Z评分), `price_vs_bb`(价格相对布林带位置), `rsi_14`(14日RSI), `rsi_28`(28日RSI), `bb_upper`(布林带上轨), `bb_lower`(布林带下轨)<br/>**动量分析维度**：`momentum_1m`(1月动量), `momentum_3m`(3月动量), `momentum_6m`(6月动量), `volume_momentum`(成交量动量), `momentum_score`(综合动量评分), `volume_confirmation`(成交量确认)<br/>**波动率分析维度**：`historical_volatility`(历史波动率), `volatility_regime`(波动率体制), `volatility_z_score`(波动率Z评分), `atr_ratio`(ATR比率)<br/>**统计套利维度**：`hurst_exponent`(赫斯特指数), `skewness`(偏度), `kurtosis`(峰度) |

## 3. 核心处理逻辑

### 3.1 行业差异化权重分配

根据行业特性和交易特征，动态调整各维度指标的权重占比：

| 行业类型 | 趋势权重 | 均值回归权重 | 动量权重 | 波动率权重 | 统计套利权重 | 核心参考标准 |
|---------|--------|-----------|--------|----------|-----------|-----------|
| **消费行业**<br/>(白酒 / 食品 / 家电) | 30% | 20% | 25% | 15% | 10% | ADX>25表示强趋势，RSI14在40-60健康区间，Z评分[-1,1]为正常波动 |
| **科技行业**<br/>(电子 / 计算机 / 高端制造) | 20% | 15% | 35% | 20% | 10% | 高波动率(15-30%)下重点关注综合动量评分与成交量确认，多月动量共振 |
| **医药行业**<br/>(创新药 / 医疗器械) | 25% | 25% | 25% | 15% | 10% | 低波动特征，RSI14/28波段操作(30-70)，赫斯特指数>0.5表示趋势延续 |
| **周期行业**<br/>(钢铁 / 有色 / 化工 / 资源) | 15% | 15% | 25% | 30% | 15% | 高波动率(20-40%)下支撑ATR比率与波动率Z评分，关注偏度与峰度极值 |
| **金融行业**<br/>(银行 / 保险 / 证券) | 35% | 20% | 15% | 15% | 15% | EMA收敛信号，ADX>25强趋势，低波动率(10-15%)特征，Z评分稳定 |
| **房地产行业**<br/>(住宅 / 物业 / 开发) | 25% | 20% | 25% | 20% | 10% | EMA系统完整，成交量确认是关键信号，监控波动率异常 |
| **公用事业**<br/>(电力 / 水务 / 燃气) | 40% | 15% | 15% | 20% | 10% | 强趋势追踪(ADX>30)，低波动(8-12%)，赫斯特指数>0.55表示强延续 |
| **通用行业**<br/>(未知行业默认) | 28% | 18% | 25% | 18% | 11% | 全市场通用技术面标准 |

### 3.2 指标标准化打分

将用户输入的原始指标，转换为 0-1 的标准化得分，用于后续加权计算：

#### 3.2.1 趋势分析维度评分

评估股票价格趋势强度与方向：

1. **ADX指数评分**（范围 0-100）：
   - ADX < 15：得分 0.3（无明显趋势）
   - 15 ≤ ADX ≤ 25：得分 0.6（趋势形成中）
   - 25 < ADX ≤ 40：得分 0.9（强趋势）
   - ADX > 40：得分 1.0（极强趋势）

2. **趋势强度评分**（0-1 数值）：
   - 直接映射：得分 = 趋势强度值（已归一化）
   - 若趋势强度 > 0.7 且 ADX > 25：额外 +0.1 bonus

3. **EMA 多头排列**（比较 EMA8, EMA21, EMA55）：
   - 完整多头（EMA8 > EMA21 > EMA55）：得分 1.0
   - 部分多头（至少满足两个 EMA 顺序）：得分 0.7
   - EMA 交错或反序（空头排列）：得分 0.3

4. **短期 / 中期趋势一致性**：
   - 短期与中期趋势同向（均为上升或下降）：得分 +0.15
   - 趋势不一致或相反：得分 -0.1

5. **缺失指标处理**：若 EMA 数据不完整，使用 ADX 和趋势强度评分替代

#### 3.2.2 均值回归维度评分

评估股票价格相对均值的偏离程度与回归机会：

1. **Z评分评分**（标准差度量）：
   - Z评分 < -2 或 > 2：得分 0.9（极度偏离，强回归信号）
   - -2 ≤ Z评分 ≤ -1 或 1 < Z评分 ≤ 2：得分 0.7（中度偏离）
   - -1 < Z评分 ≤ 1：得分 0.5（正常波动范围）
   - 0 < Z评分 < 0.5：得分 0.55（轻微上升偏离）

2. **价格相对布林带位置评分**（0-1）：
   - 价格 ≤ 下轨（price_vs_bb ≤ 0）：得分 0.8（超卖，买入机会）
   - 下轨 < 价格 < 中轨（0 < price_vs_bb < 0.5）：得分 0.6（向上回归中）
   - 中轨 ≤ 价格 ≤ 上轨（0.5 ≤ price_vs_bb ≤ 1）：得分 0.5（中性区）
   - 价格 > 上轨（price_vs_bb > 1）：得分 0.8（超买，卖出机会）

3. **RSI 评分**（RSI14 与 RSI28 综合）：
   - RSI14 < 30 或 RSI28 < 30：得分 0.8（超卖区）
   - 30 ≤ RSI14 ≤ 70 且 30 ≤ RSI28 ≤ 70：得分 = (RSI14 + RSI28) / 200（中性区间）
   - RSI14 > 70 或 RSI28 > 70：得分 0.8（超买区）
   - RSI14 与 RSI28 背离（一升一降）：额外 +0.1（潜在反转信号）

4. **布林带宽度**：
   - 布林带宽度 < 历史均值 × 0.5：得分 0.4（过度压缩，波动可能突增）
   - 历史均值 × 0.5 ≤ 宽度 ≤ 历史均值 × 1.5：得分 0.8（正常宽度）
   - 宽度 > 历史均值 × 1.5：得分 0.6（过度扩张）

#### 3.2.3 动量分析维度评分

评估价格与成交量的动量强度与确认度：

1. **多月动量综合评分**（1个月 / 3个月 / 6个月）：
   - 三个月动量均为正（向上）：得分 1.0（强上升动量）
   - 两个月动量为正：得分 0.7
   - 一个月动量为正：得分 0.5
   - 全部为负（向下）：得分 0.2（强下行动量）
   - 正负混合（动量分化）：得分 0.4

2. **综合动量评分**（已聚合的 0-1 数值）：
   - 直接映射：得分 = 综合动量评分值
   - 若综合评分 > 0.6，加权占比增加 10%

3. **成交量动量评分**（volume_momentum）：
   - 成交量动量 > 0.6：得分 0.9（量能充足）
   - 0.3 ≤ 成交量动量 ≤ 0.6：得分 0.65（量能温和）
   - 0 < 成交量动量 < 0.3：得分 0.4（量能不足）
   - 成交量动量 ≤ 0：得分 0.2（量能萎缩）

4. **成交量确认度**（volume_confirmation）：
   - 确认度 > 0.75：得分 1.0（量价齐头）
   - 0.5 < 确认度 ≤ 0.75：得分 0.8（量价协调）
   - 0.25 ≤ 确认度 ≤ 0.5：得分 0.5（量价分化中）
   - 确认度 < 0.25：得分 0.3（量价严重背离）

#### 3.2.4 波动率分析维度评分

评估市场波动率水平与体制特征：

1. **历史波动率评分**（对标行业基准）：
   - 波动率 ≤ 行业基准 × 0.7：得分 0.9（低波动，风险小）
   - 行业基准 × 0.7 < 波动率 ≤ 行业基准 × 1.2：得分 1.0（正常波动）
   - 行业基准 × 1.2 < 波动率 ≤ 行业基准 × 1.8：得分 0.65（高波动）
   - 波动率 > 行业基准 × 1.8：得分 0.3（异常高波动，风险大）

2. **波动率体制分类**（0=低, 1=中, 2=高）：
   - 体制 0（低波动）：得分 0.85
   - 体制 1（中波动）：得分 0.95（最优）
   - 体制 2（高波动）：得分 0.65

3. **波动率 Z 评分**（标准差度量）：
   - Z评分 < -1：得分 0.8（波动率偏低，可能酝酿反转）
   - -1 ≤ Z评分 ≤ 1：得分 0.9（波动率正常水平）
   - 1 < Z评分 ≤ 2：得分 0.7（波动率偏高）
   - Z评分 > 2：得分 0.4（波动率异常高企）

4. **ATR 比率评分**（ATR / 平均价格）：
   - ATR比率 < 1%：得分 0.8（极低波幅，可能盘整）
   - 1% ≤ ATR比率 < 3%：得分 0.95（适合交易）
   - 3% ≤ ATR比率 < 5%：得分 0.75（波幅扩大）
   - ATR比率 ≥ 5%：得分 0.5（波幅过大，风险高）

#### 3.2.5 统计套利维度评分

评估价格序列的统计特性与异常机会：

1. **赫斯特指数评分**（Hurst Exponent，范围 0-1）：
   - H < 0.4：得分 0.8（平均回归，适合反复交易）
   - 0.4 ≤ H ≤ 0.5：得分 0.6（随机游走，无明显趋势）
   - 0.5 < H ≤ 0.65：得分 0.85（趋势延续，适合趋势跟踪）
   - H > 0.65：得分 0.95（强趋势）

2. **偏度评分**（Skewness，衡量分布不对称性）：
   - 偏度 < -1：得分 0.7（负偏，可能下跌突发）
   - -1 ≤ 偏度 ≤ 0.5：得分 0.85（左偏或对称，分布稳定）
   - 0.5 < 偏度 ≤ 1：得分 0.8（右偏，可能上升机会）
   - 偏度 > 1：得分 0.75（强正偏，极值概率上升）

3. **峰度评分**（Kurtosis，尾部风险度量）：
   - 峰度 < 2：得分 0.85（平坦分布，极值少）
   - 2 ≤ 峰度 ≤ 4：得分 0.9（正常分布，风险可控）
   - 4 < 峰度 ≤ 8：得分 0.65（厚尾分布，极值风险上升）
   - 峰度 > 8：得分 0.4（极厚尾，黑天鹅风险大）

#### 3.2.6 缺失指标处理

- 每个维度缺失指标：使用该维度其他可用指标得分，同时降低该维度权重 15%
- 全维度指标缺失：使用中性得分 0.5，降低置信度 0.08
- 缺失指标总数 > 总指标数的 50%：使用中性得分 0.5，重新评估置信度 = 原置信度 × 0.6

### 3.3 综合得分与决策生成

1. **计算加权综合得分**：
   ```
   综合得分 = Σ(各维度得分 * 对应权重)
   范围 0~1，其中各维度权重之和 = 100%
   ```

2. **决策映射规则**：

   - `综合得分 ≥ 0.72` → **buy（买入）**：技术面强势，多维度共振，买入信号清晰
   
   - `0.48 ≤ 综合得分 < 0.72` → **hold（持有）**：技术面中性，信号不明确，观望或小额介入
   
   - `综合得分 < 0.48` → **sell（卖出）**：技术面弱势，存在风险信号，谨慎持股

3. **时间框架与信号强度调整**（可选）：
   - 若支持多周期数据：日线优势 + 周线优势 = 信号权重 +20%
   - 若仅日线数据：基础信号权重保持

### 3.4 置信度计算

置信度范围 0~1，反映决策的可靠程度，计算逻辑：

```Plain Text

置信度 = (指标覆盖率 * 0.6) + (指标一致性 * 0.4)
```

1. **指标覆盖率**：用户提供的核心指标占该行业所需核心指标的比例，越高说明信息越完整

2. **指标一致性**：所有指标得分的标准差的反向归一化，标准差越小说明指标表现越一致，决策越可靠

    - 一致性 = 1 - min (得分标准差，0.5)

    - 例：得分标准差为 0.1，一致性 = 0.9；标准差为 0.4，一致性 = 0.6

## 4. 边界约束

1. **适用范围**：仅针对 A 股市场公开可获取的股票技术面指标进行中短线择时分析，不涉及内幕信息、基本面价值判断、宏观政策解读，不支持超短线分钟线级别高频交易决策。

2. **投资提示**：输出结果仅为投资参考，不构成任何投资建议，市场有风险投资需谨慎

3. **指标限制**：仅处理合理范围内的技术指标，异常值（如成交量为负、价格为负、换手率 > 100%、涨跌幅超出 A 股 ±10%/±20% 涨跌幅限制）将直接拦截，不进行分析。

4. **行业限制**：对主流行业提供精准权重，其他行业使用通用模型，决策置信度会自动下调 0.1

## 5. 错误处理与降级策略

### 5.1 输入校验错误

|错误场景|处理方式|
|---|---|
|缺失股票行业信息|返回错误：`MISSING_INDUSTRY`，提示 "请提供股票所属行业，以匹配差异化分析权重"|
|缺失所有技术指标|返回错误：`MISSING_INDICATORS`，提示 "请提供至少 1 项技术指标进行分析"|
|指标值异常（超出合理范围）|返回错误：`INVALID_INDICATOR`，提示 "输入的指标值异常，请检查数据正确性"|
### 5.2 运行时错误与降级

|错误场景|降级策略|
|---|---|
|调用行业均值 API 失败|自动降级为使用行业通用参考标准，同时将置信度下调 0.1，在原因中提示 "无法获取最新行业均值，使用通用标准分析"|
|核心指标缺失|使用中性值补全缺失指标，同时将置信度下调 0.15，在原因中提示 "缺失部分核心指标，分析结果存在一定偏差"|
|未知行业|自动切换为通用行业权重模型，同时将置信度下调 0.1，在原因中提示 "未识别的行业，使用通用基本面模型分析"|
|网络超时 / 工具调用失败|重试 3 次（指数退避：1s/2s/4s），重试失败后执行离线分析，使用本地默认参数|
